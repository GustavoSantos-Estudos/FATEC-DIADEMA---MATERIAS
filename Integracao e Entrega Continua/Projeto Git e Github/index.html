Comandos Git e GitHub

Comandos Git

  • git init – Inicia um repositório Git na pasta.
  • git add . – Adiciona arquivos para o commit.
  • git commit -m "mensagem" – Salva alterações no repositório.
  • git status – Mostra o estado dos arquivos.
  • git log – Lista o histórico de commits.

Comandos GitHub

  • git remote add origin URL – Conecta o repositório local ao GitHub.
  • git push -u origin main – Envia alterações para o GitHub.
  • git pull – Baixa alterações do repositório remoto.
  • git clone URL – Copia um repositório remoto para o computador.
